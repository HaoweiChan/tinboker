# 選題、轉化漏斗、趨勢監測與防限流

## 選題三維度（決定 80% 成敗）

寫作技巧只佔 20%，選題本身佔 80%。用三維度篩選候選話題，不要漫無章法發私人日記：

- **相關性 Relevance**——落在自己的領域守備範圍內。
- **時效性 Timeliness**——當下正在被討論的時事/數據。
- **共鳴度 Resonance**——多數人有切身經驗或情緒共鳴。

適度負面情緒（合理抱怨、憤怒、揭露不公）會引發即時情緒對撞，擴散速度通常最快。

## 財經利基的四大高互動切角

1. **降低門檻的日常經驗**——非 A 即 B 的簡單選擇（例如「每天少喝一杯咖啡，30 年後真的能買房嗎？」），降低回覆門檻。
2. **知識解答型**——專家解惑、產業避雷清單、節稅整理、反詐騙案例。長尾效應強，會持續被搜尋/收藏。
3. **日常反差驚奇**——強烈反差情境（例如「月薪三萬靠被動收入年領百萬的作息表」），引導呼朋引伴圍觀。
4. **情緒與立場對撞**——針對熱門時事（高股息 ETF 是否是騙局、買房 vs 租房）引導兩派論點對抗，帶動留言蓋樓。

用「種草分享」取代專業說教：與其貼精美分析圖表，不如分享自己「投資慘賠復盤」或「30 歲前理財真實盲區」，搭配隨手截圖，降低業配感。

## 非直球對決的轉化漏斗

Threads 用戶對硬廣告／業配極度反感，直接放購買連結流量會很差。把 Threads 當「流量拉霸機」：

- 平日：低壓力隨手記錄想法/觀察，維持帳號溫度。
- 爆文發生時：**立刻**在該貼文留言區補上電子報／播客訂閱連結（不要放在主文）。
- 平時：把轉化放在**個人自介欄**（PPP+C 框架：Profile 定位、Proof 佐證、Promise 承諾、+ CTA），把陌生流量導入私域長期經營。

## 趨勢監測（由輕到重）

- **平台原生**：Threads 搜尋頁的「全台前五大熱門話題」AI 摘要；動態牆上帶「超夯」標籤的貼文＝當下演算法正在推播的風口。
- **第三方輿情工具**：QSearch Trend+（每 4 小時更新熱門話題/關鍵字，可回溯 2 年聲量）、Threadslytics（排程發文、破圈率分析）。這類商業工具比自建爬蟲省事，適合先用這個驗證需求。
- **自建自動化採集**：技術上可行（Meta 官方 API 需 App Review + OAuth，限制嚴格；第三方爬蟲免登入但涉及違反 Threads 條款與帳號封鎖風險），這裡不內建爬蟲腳本。若要做，先確認合規與 ToS 風險，抓取頻率保守（每日 1 次即可），只抓自己帳號或公開可見內容做選題參考，不做大規模採集。

## 防限流（Shadowban）自救三步驟

1. 檢查近期新增粉絲，若有大量無頭像/純轉發垃圾帳號，手動封鎖。
2. 發一則有「人情溫度」的生活抱怨或寵物貼文，通常能快速重建演算法權重。
3. 若持續低迷，去當天熱門財經 KOL 貼文下方留下有深度、幽默、與自己領域高度相關的留言（海巡法），藉由對方的高流量池把陌生用戶導回自己主頁。
