# 寫作模板與播客轉文字公式

## 語氣總則

Threads 是「草稿心態」與「閒聊語境」，不是 IG 的精緻感。刻意保留粗糙感（隨手感）比精修文案更有傳播力。**絕對禁止**：好為人師、說教、官腔、行銷感強烈的詞彙。夜店理論——先給「解決問題後的理想畫面」，不要先給「解決問題的枯燥步驟」。

## 模板 A：結構驅動型（理財乾貨／產業避雷／財報拆解）

用於：完讀率、關注轉化率、點擊率導向的內容。

```
[黃金 Hook 開頭]（≤15 字，禁止折疊，製造認知衝突或強烈好奇）

[共鳴段／代入感]（1–2 句勾勒讀者切身痛點或焦慮場景）

[觀點翻轉／核心乾貨]（短句或條列，3–5 點封頂）

[低門檻二選一提問 / 引導轉發金句]（3 秒內能回答）
```

## 模板 B：共鳴驅動型（投資挫折復盤／心路歷程／財富觀重塑）

用於：停留時間、轉發數、收藏數導向的內容。

```
[平穩且真實的場景切入]（不用花俏標題，直接說故事）

[情緒逐步累積]（真實掙扎/盲區/彎路，展現脆弱感與真實人設）

[頓悟時刻／觀點昇華]（不說教，只分享自己學到的一課）

[留白結尾]（一句能引發讀者內心投射的溫慢結論，不強推硬結論）
```

## 播客轉文字三步驟

一集 45–60 分鐘的播客，轉成 150–250 字的 Threads 貼文：

1. **提煉單一核心矛盾**——不要總結整集。一集談 5 個主題，一則貼文只說 1 件事。
   - ❌「這集聊了台積電財報、美股大盤、以巴衝突、還有理財建議」（訊息過載）
   - ✅「台積電毛利率創新高，為什麼散戶不該現在追高？」（聚焦單一爭議）
2. **套用夜店理論重塑開頭**——先講讀者最在乎的畫面，不要先講步驟。
   - ❌「今天來教大家怎麼看現金流量表的三個指標」
   - ✅「很多公司表面上年賺百萬，私底下連下個月薪水都發不出來。關鍵在...」
3. **導入無痛閱讀排版**——手機一行約 18–20 個中文字，長句必須切斷、一句一換行；轉折處或條列前可用 emoji（💡⚠️🔥），全篇 ≤3 個。

## 格式硬規則（寫完必檢查）

- 第一行 ≤15 字，不帶贅字，有畫面感或衝突點。
- 一段一概念，段落間留空行。
- 條列 3–5 點封頂，超過就拆成下一篇或留言區置頂。
- 結尾提問要有「選項感」，不要「哲學思考」。
  - ❌「你覺得台積電會漲到一千五嗎？」（需要分析，回覆率低）
  - ✅「你現在手上的台積電，是偏向繼續加碼，還是落袋為安？」（非 A 即 B）
- 主文 150–250 字。

## LLM 改寫 Prompt（有逐字稿/筆記時直接套用）

```
你是一位殿堂級的社群數據分析師與財經自媒體文案大師。現在我會提供一段【財經播客的筆記/逐字稿內容】，
請改寫為符合 Threads 平台高傳播演算法的爆文，嚴格遵循：

【語調】極具人味、閒聊語境、草稿感。禁止說教/官腔/行銷感詞彙。
【開頭】第一行 ≤15 字，用認知衝突、痛點劇透或乾貨誘餌製造強烈吸引力。
【排版】短句呼吸感，一句一換行，段落間留空行，嚴禁大塊文字。條列僅限 3 點以內。
【架構】提供兩種版本：
  版本 A（結構驅動）：強 Hook + 切身焦慮 + 極簡乾貨 + 低門檻二選一提問。
  版本 B（共鳴驅動）：故事場景切入 + 挫折與掙扎（展現脆弱）+ 溫暖觀點昇華。
【字數】主文 150–250 字。

播客筆記內容：
[在此貼上原始筆記]
```
