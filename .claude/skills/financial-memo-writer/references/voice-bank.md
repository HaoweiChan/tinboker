# Voice Bank — investment-writing style reference

Style distillation for the `financial-memo-writer` skill: lexicon, analogy techniques, and
representative excerpts from a corpus of in-depth zh-TW investment writing. Use these as few-shot
calibration — match the *register and rhythm*, do not copy verbatim into new pieces.

---

## 1. Signature vocabulary & phrases (the lexicon)

**Mottos / framing tics**
- 撇除雜訊，回歸邏輯 · 拆解底層的商業邏輯 · 用研究與邏輯，取代情緒與跟風

- 這代表什麼？/ 這意味著什麼？ · 你看到關鍵了嗎？ · 你可能會問…
- 先講結論： · 老實說… · 說到底… / 說穿了 / 說白了 / 講白了 · 換句話說…

## 2. Analogies (technique bank — invent fresh ones in the same spirit)

| Concept | Analogy |
|---|---|
| GPU vs CPU | 一萬名士兵 vs 十位大學教授；GPU 是肌肉，CPU 是神經系統 |
| 獨立發電商 (IPP) | 沒有底薪、純靠業績抽成的業務員 / 發電傭兵 |
| 公用事業 (IOU) | 自來水公司 / 高速公路收費站 |
| 電力批發市場 | 每日海鮮拍賣場 |
| Chiplet vs Monolithic | 樂高積木 vs 一整塊大石頭 |
| NVLink vs PCIe | 私人高鐵 vs 省道 |
| 晶圓級晶片 (記憶體牆) | 超快廚師但食材在隔壁倉庫 → 把倉庫搬進廚房 |
| 能隙 (bandgap) | 捷運進站閘門的票價（電子=乘客，電壓=口袋裡的錢） |
| 跨資料中心互連 (DCI) | 蓋八棟 50 層樓，用「空橋」連成一棟 |
| LNG 出口站 | 天然氣的超級冷凍工廠兼收費公路 |
| 煉油廠 | 大型麵包店（原油=麵粉，成品油=蛋糕） |
| CDN / WAF | 地方快遞員 / 店門口的保鑣 |
| 雲端資料傳出費 (egress) | 加州旅館「You can check out any time you like, but you can never leave」 |
| Fed 收緊 | 在派對漸入佳境時把雞尾酒盆收走的人 |
| 平台 vs 算力 | 房東 vs 所有租客都要打交道的水電公司 |
| 散熱物理 | 散熱沒做好 → 元件變熱 → 電阻上升 → 損耗變大 → 產生更多熱（惡性循環） |

Intro phrases: 你可以想像成… · 打個比方， · 比喻時間： · 你可以把它想成…

## 3. Opening patterns (representative)

- **(overturn-a-belief, deep-dive)** 「在過去數年，整個科技界和華爾街只有一個共同信仰：『得 GPU
  者得天下』…但在 2026 年的今天…最缺貨、最讓人頭痛的運算瓶頸…變成了 CPU。」
- **(cinematic scene)** 「2024 年 9 月，三哩島核電廠，這家公司的執行長站在這個曾經因為 1979 年
  美國史上最嚴重核災而舉世聞名…的地方，向全世界宣布了一項重要的決定。」
- **(shock data point)** 「最新一代的 AI 機櫃，單櫃功耗已達約 120kW；而下個旗艦機架，單櫃功耗
  更將直接衝上 600kW。」
- **(counterfactual + aphorism)** 「如果你在今年年初問任何一位交易員『Fed 下一步是什麼』，得到的
  答案幾乎是同樣：降息。市場是一台會自我強化的機器，當所有人都站在同一邊，敘事就會變成信仰。」
- **(follow-the-money)** 「投資市場裡有一條不成文的規矩：『跟著聰明的錢走』…但到了 2026 年的
  今天，市場上最大、也最激進的『創投』，其實是美國政府。」
- **(weekly-note greeting + mood)** 「歡迎來到本期的週度市場筆記。呼，這個星期是真的瘋狂。四大
  巨頭在同一天、同一時間公佈財報，這節奏簡直是想讓全球的分析師集體失眠。」
- **(column, price punch)** 「599 美元的 MacBook，599 美元的 iPhone。過去十八年，蘋果的哲學很
  簡單：我們只做高端，讓別人去搶那些不賺錢的市場。」

## 4. Closing patterns (representative) — never end on data

- **(終局悖論, deep-dive)** 「經濟學有一個鐵律：『解決高價最好的解藥，就是高價本身。』…你現在
  買入的 IPP，買的並不是一個『永久免疫競爭的無敵護城河』，而是一個『具有明確時效性的時間差』。」
- **(paradigm-elevation + bookend)** 「愛迪生與特斯拉在 140 年前的電流戰爭裡留下了關於效率與傳輸
  的課題，今天這道課題在 AI 機架上得到了最極致的解答…這兩條曲線誰跑得快，就是未來三到五年值得
  留意的一場賽跑。」
- **(risk-humble aphorism)** 「這個產業的故事很動人，但它的本質從來沒變過…再動人的成長敘事，都
  改變不了『擴產 → 過剩 → 殺價』的鐵律。最危險的時刻，永遠是市場一致認為『這次不一樣』的時候。」
- **(矛與盾 strategy close)** 「把高質量的 AI 龍頭當作你的矛，把金融、能源、短債與黃金當作你的
  盾…確保無論曲終時是哪個調，自己都還安全地站在舞池裡，才是最優雅的姿態。」
- **(column, wry one-liner)** 「這一次會一樣嗎？沒有人知道。但至少現在我們知道蘋果押在哪裡：不是
  雲端，是口袋。」
- **(weekly-note forward question)** 「下次看到『XX 需求腰斬』這種標題時，或許可以先問一句：腰斬
  的是哪一層？為什麼？這對其他層級有什麼影響？答案通常比標題有趣得多。」

## 5. Risk / humility / disclaimer excerpts (representative)

- **Scope disclaimer (industry deep-dive)** 「這是一篇純產業分析，而非個股推薦。我不會提供目標價，
  也不會討論任何個股的合理本益比…我更願意做的事，是幫你建立一張產業地圖。」
- **Valuation hedge** 「（一如既往，所有的『估值』其實包含了大量的推算，不需要太過深究具體數字，
  理解背後框架更重要。）」
- **利益申報 (personal-bias disclosure)** 「我必須坦承，這篇報告…可能帶有我個人不可避免的『看多
  偏見』…甚至請你主動對我文中的樂觀預期『打個折扣』。」
- **Kill Switch / sell trigger** 「什麼情況下你該賣（整個板塊的 Kill Switch）：短期殺手（需求端）：
  雲端巨頭資本支出指引突然大幅下修（例如某季財報大砍 Capex 指引 20% 以上）。」
- **Position-sizing** 「它不是你投資組合中的『核心持倉』，而是一張『看漲選擇權』…願意承受 50%
  以上的下行風險來換取未來 5–10 年的爆發性上行空間，那麼可以作為 5–10% 的衛星配置。」
- **Fairness caveat** 「必須特別澄清：指出這些企業的短處，並不代表它們不好…它們都是這場科技盛宴
  中無庸置疑的受益者。」
- **Volatility-as-entry-fee** 「波動（Volatility）是每一位權益投資者必須支付的『入場券』…你不可能
  在拒絕接受 10% 甚至更多回撤的同時，卻期待獲得 50% 甚至 100% 的回報。」
- **Humility** 「我不是神，沒有人會能夠完全預測利率和市場走勢的下一步，但我相信看完這篇過後，你
  會對現在的經濟數據有更清楚的理解。」

## 6. Representative full-voice excerpts (the gold standard)

1. **(teaching-over-picks)** 「在我寫過的深度分析中，我從來不會直接叫大家在什麼價位買入，我提供
   的是每一家公司背後極其深層的運作邏輯…我只是為大家攤開地圖。如果你最後獲利了，所有的功勞都
   屬於你自己。」

2. **(resilience philosophy)** 「作為專業投資者，我們最核心的任務不是『猜對』，而是『確保自己的
   投資組合在任何劇本下都能生存』…每一個標的都有明確的買入邏輯與離場條件，即便最壞的情況發生，
   我依然能睡得著覺。」

3. **(認知邊界 mission)** 「與其追求『神預測』，不如持續推高自己的『認知邊界』…我們永遠抓不到
   最極端的轉折點。只要對得起自己的分析，堅持用邏輯行事，長久下來，正收益是認知的必然結果。」

4. **(decode-the-surface)** 「換句話說，你多花一倍的錢買一台可以『一次印好』的機器，但印出來的
   單位成本可能還比『印兩次的舊機器』貴。這就是為什麼台積電的態度是：謝謝，暫時不用了。」

5. **(翻譯成投資語言)** 「那句『我們的研發團隊還在持續從現有 EUV 挖出好處』，翻譯成投資語言
   就是：『我們的 ROIC 沒你們想的那麼依賴下一代機台。』」

6. **(data with a so-what)** 「它只賣了不到三成的量，卻拿走了超過四成的錢。」

7. **(layer-disambiguation)** 「事實上，這兩家機構都沒說錯，只是看的是電力架構中不同的『樓層』…
   兩者講的根本是不同樓層的事，並不衝突。」

8. **(decision-framework refusal)** 「我不會給你一個標準答案，因為你的劇本決定了你的選擇。請對號
   入座：劇本一…劇本四：你不相信這個故事選擇觀望…放棄一個看不懂的好機會，永遠比參與一個看不懂
   的爛機會划算。」

9. **(quoted maxim + own gloss)** 「我很喜歡一句名言：『悲觀者往往聽起來很聰明，但最終賺錢的通常
   是樂觀者。』這並不是叫大家盲目樂觀，而是作為一名股票投資者，你的底層邏輯必須是相信未來的。」

10. **(skepticism / AI-washing)** 「當然，我們必須保持一貫的理性與懷疑。在這波 2026 年的科技業
    裁員潮中，真的全是因為『AI 太強大』嗎？…這就是所謂的『AI-washing』。」
